Problem description

You are given a long string. Your task is to find the smallest and largest three-character substrings, 
in lexicographical order, in the given string. For example, the string “abracadabra” 
contains 9 three-character substrings: "abr","bra","rac","aca","cad","ada","dab","abr" and "bra." 

The smallest of these is “abr” and the largest is “rac”.
Note that lexicographical order is the order of words in a dictionary. 
For example, “cat” is smaller than “dog” because “c” is smaller than “d”. “has” is smaller than “hat” 
because the first two characters are the same, and “s” is smaller than “t”.

Input

Test cases will be provided in the following multiline text format, using only lower case alphabetic characters. The first line contains one integer, C, which is the number of test cases that will follow.From the second line onwards, one test case is provided per line and each test case contains one string.

Limits
Number of test cases: 1 <= C <= 50.
3 <= Input string’s length < 100.
Output

For each test case, output the result in the following format:

Case #k: S_small S_large

where k is the index of the test case, starting from 1, S_small is the smallest three-character substring and S_large is the largest three-character substring.

Sample input (task1_small_sample.input.txt)
7
aaaa
zzzz
aaaz
zaaa
xyz
xyzabc
abracadabra

Output for sample input (task1_small_sample.output.txt)
Case #1: aaa aaa                                                                                                                                               
Case #2: zzz zzz
Case #3: aaa aaz
Case #4: aaa zaa
Case #5: xyz xyz
Case #6: abc zab
Case #7: abr rac
